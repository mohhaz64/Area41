######################################
# Compilation with terminal (MAC OS) #
######################################

1. After right clicking 'source' folder click on "Open New Terminal Window at Folder"
2. To compile java files first run the command "javac *.java"
3. To run the game run command "java Main"


#############################################
# Compilation with CMD/PowerShell (Windows) #
#############################################

1. Open project to 'source' folder in Windows Explorer
2. Having held shift, right click on the Window Explorer window and select "Open Command Window Here"
3. To compile java files first run the command "javac *.java"
4. To run the game run command "java Main"
